Adobe DPS Folio Author tools for Wordpress.

Folios Author is a plugin for Wordpress that allows publishers to create HTML articles and folios directly from Wordpress. Using this plugin, publishers can streamline the creation of their content on Digital Publishing Suite apps across multiple devices.
